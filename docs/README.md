# Peach Documentation

Documentation for **Peach**, an adaptive pitch discrimination training app for iOS.

## Directory Overview

| Directory | Description |
|---|---|
| [`arc42/`](arc42/index.md) | Architecture documentation following the [arc42](https://arc42.org) template — introduction & goals, constraints, building blocks, runtime views, quality requirements, and more. |
| [`planning-artifacts/`](planning-artifacts/) | Product planning documents: [PRD](planning-artifacts/prd.md), [epics & stories](planning-artifacts/epics.md), [UX design spec](planning-artifacts/ux-design-specification.md), [architecture](planning-artifacts/architecture.md), [glossary](planning-artifacts/glossary.md), and validation reports. |
| [`implementation-artifacts/`](implementation-artifacts/) | Per-story implementation documents tracking each feature from Epic 1 through Epic 30, plus hotfixes, infrastructure tasks, and a [future work](implementation-artifacts/future-work.md) backlog. |
| [`brainstorming/`](brainstorming/) | Brainstorming session notes from product discovery and ideation. |
| [`test-artifacts/`](test-artifacts/) | Test strategy reviews and quality documentation. |
| [`bmb-creations/`](bmb-creations/) | Artifacts created by the BMAD Module Builder, including the music domain expert agent. |
| [`claude-audit/`](claude-audit/index.md) | Human-readable, condensed summaries of every Claude Code session during development — no tool output, just the conversation flow. Generated by [`bin/claude-audit.py`](../bin/claude-audit.py). |
| [`reports/`](reports/) | Analytical reports on the development process. |
